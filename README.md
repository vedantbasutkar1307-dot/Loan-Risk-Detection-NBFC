# Early Loan Risk Detection for NBFCs (Piramal Finance Use Case)

## Overview
This project predicts loan defaults using machine learning and visualizes risk insights using Power BI.
---

## Tech Stack
* Python (Pandas, NumPy, Scikit-learn)
* Power BI
* Jupyter Notebook
---

## Key Results
* Improved risky customer detection (Recall) from **3% → 63%**
* Built interactive dashboard for decision-making
---

## Dashboard Features
* Risk segmentation (High vs Low Risk)
* Risk probability distribution
* Model performance visualization
* High-risk customer identification
---

## Business Impact
* Helps NBFCs reduce loan defaults
* Enables data-driven lending decisions
---
